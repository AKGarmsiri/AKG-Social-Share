AKG Social Share Instructions

The AKG Social Share plugin is a simple way to let your visitors share your WordPress posts on social media.

Installation

To install the plugin, follow these steps:

  1. Download the plugin from the WordPress plugin repository or from the GitHub repository.
  2. Upload the plugin to your WordPress site by navigating to Plugins > Add New > Upload Plugin, then selecting the .zip file you downloaded.
  3. Activate the plugin by navigating to Plugins and clicking "Activate" under AKG Social Share.

Usage

To display the social media sharing links on your WordPress posts or pages, use the following shortcode:

[akg_social_share]

You can also customize which social media links are displayed using the following attributes:

  • facebook - Set to "yes" to display the Facebook sharing link, or "no" to hide it.
  • twitter - Set to "yes" to display the Twitter sharing link, or "no" to hide it.
  • linkedin - Set to "yes" to display the LinkedIn sharing link, or "no" to hide it.
  • whatsapp - Set to "yes" to display the WhatsApp sharing link, or "no" to hide it.
  • instagram - Set to "yes" to display the Instagram sharing link, or "no" to hide it.

For example, to display the Facebook, Twitter, and LinkedIn sharing links only, use the following shortcode:

[akg_social_share facebook="yes" twitter="yes" linkedin="yes" whatsapp="no" instagram="no"]

You can place the shortcode anywhere in your post or page content to display the social media sharing links.

Customization

If you'd like to customize the look and feel of the social media sharing links, you can use CSS to style them however you like. The social media icons are generated using Font Awesome, so you can use any of the Font Awesome icons to replace the default ones.

Here's an example CSS snippet to change the background color of the social media icons:

.akg-social-share a {
    background-color: #333;
}

Feel free to experiment with different CSS styles to make the social media sharing links fit your website's design.
